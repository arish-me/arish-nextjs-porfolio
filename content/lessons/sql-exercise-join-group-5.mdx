## Exercise: Multiple Aggregates

Get count, average, and max marks per course.

## Solution

```sql
SELECT c.course_name,
  COUNT(*) AS students,
  AVG(s.marks) AS avg,
  MAX(s.marks) AS max
FROM students s
JOIN courses c ON s.course_id = c.id
GROUP BY c.course_name;
```
