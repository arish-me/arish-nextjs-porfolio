## Brakeman Security Scanner

```yaml
jobs:
  security:
    runs-on: ubuntu-latest
    
    steps:
      - uses: actions/checkout@v4
      
      - uses: ruby/setup-ruby@v1
        with:
          ruby-version: '3.2'
          bundler-cache: true
      
      - name: Run Brakeman
        run: bundle exec brakeman -q -w2
```

## Brakeman Options

```bash
# Quick scan
bundle exec brakeman -q

# Only high confidence warnings
bundle exec brakeman -w2

# Output formats
bundle exec brakeman -f json -o brakeman-output.json
bundle exec brakeman -f html -o brakeman-report.html
```

## Configuration

```yaml
# config/brakeman.yml
---
:skip_checks:
  - CheckRender

:skip_files:
  - lib/legacy/

:safe_methods:
  - :my_safe_method
```

## Bundler Audit

Check for vulnerable gems:

```yaml
- name: Run bundler-audit
  run: |
    gem install bundler-audit
    bundle-audit check --update
```

## Combined Security Job

```yaml
security:
  runs-on: ubuntu-latest
  steps:
    - uses: actions/checkout@v4
    - uses: ruby/setup-ruby@v1
    
    - name: Brakeman
      run: bundle exec brakeman -q -w2
    
    - name: Bundle Audit
      run: bundle exec bundle-audit check --update
```
