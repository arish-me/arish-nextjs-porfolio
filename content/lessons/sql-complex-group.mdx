## Multiple Table Grouping

```sql
SELECT d.dept_name, c.course_name, COUNT(*) AS students
FROM departments d
JOIN courses c ON d.id = c.dept_id
JOIN students s ON c.id = s.course_id
GROUP BY d.dept_name, c.course_name;
```
