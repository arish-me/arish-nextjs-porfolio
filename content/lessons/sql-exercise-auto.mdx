## Exercise: Auto ID Table

Create table with auto-incrementing ID.

## Solution

```sql
CREATE TABLE messages (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  content TEXT
);

INSERT INTO messages (content) VALUES ('Hello');
INSERT INTO messages (content) VALUES ('World');
```
