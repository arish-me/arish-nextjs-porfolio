## COALESCE Function

Returns first non-NULL value.

```sql
SELECT COALESCE(phone, email, 'No Contact') AS contact
FROM students;
```

Checks phone first, then email, then default.

## With Multiple Columns

```sql
SELECT name,
  COALESCE(nickname, name) AS display_name
FROM students;
```
