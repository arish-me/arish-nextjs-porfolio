## Sample Database

Throughout this course, we'll use a `students` table:

```sql
-- Sample data structure
CREATE TABLE students (
    id INTEGER PRIMARY KEY,
    name TEXT,
    age INTEGER,
    course TEXT,
    marks INTEGER,
    grade TEXT
);
```

This gives us a realistic dataset to practice with!
