## IS NULL

Find rows with missing values.

## Syntax

```sql
SELECT * FROM students
WHERE phone IS NULL;
```

## Example

Find students without phone numbers:

```sql
SELECT * FROM students
WHERE phone IS NULL;
```

## Multiple NULL Checks

```sql
SELECT * FROM students
WHERE phone IS NULL
OR email IS NULL;
```
