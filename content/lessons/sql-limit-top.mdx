## Getting Top Results

Combine ORDER BY and LIMIT.

```sql
SELECT * FROM students
ORDER BY marks DESC
LIMIT 10;
```

Returns top 10 highest marks.

## Top Performers

```sql
SELECT name, marks FROM students
ORDER BY marks DESC
LIMIT 5;
```
