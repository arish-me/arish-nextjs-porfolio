## Exercise: Marks Up To 75

Find students with marks of 75 or less.

## Solution

```sql
SELECT * FROM students
WHERE marks <= 75;
```

## Practice More

```sql
-- Age 12 or below
SELECT * FROM students WHERE age <= 12;

-- Marks up to 60
SELECT * FROM students WHERE marks <= 60;
```
