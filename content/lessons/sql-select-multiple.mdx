## Selecting Multiple Columns

Separate columns with commas.

## Syntax

```sql
SELECT name, age FROM students;
```

### Result

| name | age |
|------|-----|
| Alice | 14 |
| Bob | 12 |
| Carol | 15 |

## More Columns

```sql
SELECT name, course, marks, grade FROM students;
```
