## Column Arithmetic

```sql
SELECT name, marks, marks + 10 AS bonus_marks FROM students;

SELECT name, marks * 1.1 AS curved_marks FROM students;
```

## Operators

| Operator | Meaning |
|----------|---------|
| + | Addition |
| - | Subtraction |
| * | Multiplication |
| / | Division |
| % | Modulo |
