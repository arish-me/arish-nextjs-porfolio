## String Concatenation

Combine columns into one.

## SQLite / MySQL

```sql
SELECT name || ' - ' || course AS student_info
FROM students;
```

## Using CONCAT

```sql
SELECT CONCAT(name, ' (', age, ')') AS display
FROM students;
```
