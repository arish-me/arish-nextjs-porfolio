## Common Data Types

| Type | Description |
|------|-------------|
| INTEGER | Whole numbers |
| TEXT/VARCHAR | Text strings |
| REAL/FLOAT | Decimal numbers |
| DATE | Date values |
| BOOLEAN | True/False |

```sql
CREATE TABLE products (
  id INTEGER,
  name VARCHAR(100),
  price REAL,
  created DATE
);
```
