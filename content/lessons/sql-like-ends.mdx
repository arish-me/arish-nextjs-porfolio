## Ending With a Pattern

Use % at the beginning.

## Syntax

```sql
SELECT * FROM students
WHERE name LIKE '%er';
```

Returns names ending with 'er': Peter, Oliver...

## Examples

```sql
-- Names ending with 'a'
SELECT * FROM students
WHERE name LIKE '%a';

-- Courses ending with 'Script'
SELECT * FROM students
WHERE course LIKE '%Script';
```
