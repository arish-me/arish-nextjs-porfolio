## Project Task 6: Advanced Branching Practice

Practice merging and rebasing with your portfolio project.

## Part 1: Merge Practice

### Step 1: Create Feature Branch

```bash
cd portfolio-project
git checkout main
git pull origin main
git checkout -b feature/testimonials
```

### Step 2: Add Testimonials Section

Add to `index.html`:

```html
<section id="testimonials">
    <h2>Testimonials</h2>
    <div class="testimonial">
        <p>"Great work and attention to detail!"</p>
        <cite>- Happy Client</cite>
    </div>
</section>
```

Add to `css/style.css`:

```css
#testimonials {
    padding: 2rem;
    background: #f0f0f0;
}

.testimonial {
    border-left: 4px solid #333;
    padding-left: 1rem;
    margin: 1rem 0;
}
```

```bash
git add .
git commit -m "Add testimonials section"
```

### Step 3: Create Conflict Situation

While on feature branch, switch and make conflicting change:

```bash
# Make change on main
git checkout main
echo "/* Main branch update */" >> css/style.css
git add css/style.css
git commit -m "Update styles on main"

# Go back to feature
git checkout feature/testimonials
```

### Step 4: Merge with Conflict Resolution

```bash
git merge main
# CONFLICT in css/style.css

# Resolve the conflict manually
# Keep both changes

git add css/style.css
git commit -m "Merge main into feature/testimonials"
```

## Part 2: Rebase Practice

### Step 5: Create Another Feature Branch

```bash
git checkout main
git checkout -b feature/contact-info
```

### Step 6: Make Multiple Commits

```bash
# Commit 1
echo "<section id='contact'><h2>Contact</h2></section>" >> index.html
git add index.html
git commit -m "Add contact section"

# Commit 2
echo "#contact { padding: 2rem; }" >> css/style.css
git add css/style.css
git commit -m "Add contact styles"

# Commit 3
echo "// Contact form validation" >> js/main.js
git add js/main.js
git commit -m "Add contact form JS"
```

### Step 7: Interactive Rebase to Squash

```bash
git rebase -i HEAD~3
```

Change to:
```
pick abc123 Add contact section
squash def456 Add contact styles
squash ghi789 Add contact form JS
```

New commit message:
```
Add complete contact section

- HTML structure
- CSS styling
- JavaScript validation
```

### Step 8: Rebase onto Updated Main

```bash
git checkout main
# Make a new commit
echo "/* Another update */" >> css/style.css
git add . && git commit -m "Style update"

git checkout feature/contact-info
git rebase main
```

## Part 3: Complete the Project

### Step 9: Merge Feature to Main

```bash
git checkout main
git merge feature/testimonials
git merge feature/contact-info
git push origin main
```

### Step 10: Clean Up

```bash
git branch -d feature/testimonials
git branch -d feature/contact-info
```
