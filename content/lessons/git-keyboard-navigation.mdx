## Keyboard Navigation

You can navigate through this course efficiently using keyboard shortcuts.

## Navigation Shortcuts

### Moving Between Lessons

- **→** or **N** - Go to the next lesson
- **←** or **P** - Go to the previous lesson

### Quick Actions

- **Esc** - Return to course overview
- **Space** - Scroll down the page
- **Shift + Space** - Scroll up the page

## Why Use Keyboard Shortcuts?

Using keyboard shortcuts helps you:

1. **Save Time** - Navigate faster without reaching for your mouse
2. **Stay Focused** - Keep your hands on the keyboard while coding
3. **Build Habits** - Keyboard shortcuts are essential for developers

## Practice

Try using the keyboard shortcuts now:

1. Press **→** or **N** to go to the next lesson
2. Press **←** or **P** to come back here

The more you practice, the more natural it becomes!

## Pro Tip

Many developer tools use similar keyboard shortcuts:

- **VS Code** - Uses Ctrl/Cmd + keys
- **Terminal** - Uses arrow keys for history
- **Git commands** - Can be aliased for speed

Get comfortable with keyboard navigation early - it's a skill that will serve you throughout your development career!
