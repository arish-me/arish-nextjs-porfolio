## Exercise: Multiple FKs

Create table with two foreign keys.

## Solution

```sql
CREATE TABLE reviews (
  id INTEGER PRIMARY KEY,
  product_id INTEGER,
  user_id INTEGER,
  rating INTEGER,
  FOREIGN KEY (product_id) REFERENCES products(id),
  FOREIGN KEY (user_id) REFERENCES users(id)
);
```
