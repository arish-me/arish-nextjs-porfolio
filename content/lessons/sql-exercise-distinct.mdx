## Exercise: Unique Grades

Get all unique grades from students.

## Solution

```sql
SELECT DISTINCT grade FROM students;
```

## Practice

```sql
-- Unique ages
SELECT DISTINCT age FROM students;

-- Unique course-grade combinations
SELECT DISTINCT course, grade FROM students;
```
