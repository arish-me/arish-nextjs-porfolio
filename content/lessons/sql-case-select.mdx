## CASE Expression

```sql
SELECT name, marks,
  CASE
    WHEN marks >= 90 THEN 'Excellent'
    WHEN marks >= 80 THEN 'Good'
    WHEN marks >= 70 THEN 'Average'
    ELSE 'Needs Work'
  END AS performance
FROM students;
```

## Simple CASE

```sql
SELECT name,
  CASE grade
    WHEN 'A' THEN 'Top'
    WHEN 'B' THEN 'Good'
    ELSE 'Other'
  END AS tier
FROM students;
```
