## Excluding Contains Pattern

```sql
-- Courses NOT containing 'y'
SELECT * FROM students
WHERE course NOT LIKE '%y%';
```

## Example

```sql
-- Names NOT containing 'ar'
SELECT * FROM students
WHERE name NOT LIKE '%ar%';
```
