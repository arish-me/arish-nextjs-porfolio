## Alphabetical Order

Sort text columns alphabetically.

```sql
SELECT name FROM students
ORDER BY name;
```

Returns A-Z order: Alice, Bob, Carol...

## With Other Columns

```sql
SELECT name, course FROM students
ORDER BY course, name;
```
