## Exercise: Count Unique Values

Count the number of different grades.

## Solution

```sql
SELECT COUNT(DISTINCT grade) FROM students;
```

## Practice

```sql
-- Count unique ages
SELECT COUNT(DISTINCT age) FROM students;

-- Count unique ages of Python students
SELECT COUNT(DISTINCT age) FROM students
WHERE course = 'Python';
```
